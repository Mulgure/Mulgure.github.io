排序算法总结

发表于 2017-07-19   |  

本文主要是对阮一峰老师文章排序算法的补充与总结

课本上常见的排序算法有冒泡排序 选择排序 插入排序 合并排序 快速排序 希尔排序 前五种排序方法的算法与函数见上文即可

希尔排序

希尔排序是插入排序的一种 它是针对直接插入排序算法的改进 该方法又称缩小增量排序

对于n个待排序的数列 取一个小于n的整数gap(gap被称为步长)将待排序元素分成若干个组子序列 所有距离为gap的倍数的记录放在同一个组中 对各组内的元素进行直接插入排序 这一趟排序完成之后 每一个组的元素都是有序的 然后减小gap的值 并重复执行上述的分组和排序 重复这样的操作 当gap=1时 即所有记录放在同一组中进行直接插入排序为止 整个数列就是有序的

希尔排序实质上是一种分组插入方法

下面是具体代码

function shellSort(myArray){
    var gaps=[5,3,1];
    var res=[];
    for(var g=0;g<gaps.length;g++){
        for(var i=gaps[g];i<myArray.length;i++){
            var temp=myArray[i];
            for(var j=i;j>=gaps[g]&&myArray[j-gaps[g]]>temp;j-=gaps[g]){
                myArray[j]=myArray[j-gaps[g]];
            }
            myArray[j]=temp;          
        }      
    }  
    console.log(myArray)  
}
var myArray=[6,0,2,9,3,5,8,0,5,4];
shellSort(myArray);

时间复杂度

冒泡排序：O(n^2) 稳定

选择排序：O(n^2) 稳定

插入排序：平均O(n*log2n) 最差O(n^2) 不稳定

合并排序 O(n*logn) 稳定

快速排序 平均O(n*log2n) 最差O(n^2) 不稳定

希尔排序情况较复杂 与间隔序列有关

空间复杂度

冒泡排序：O(1)

选择排序：O(1)

插入排序：O(1)

合并排序 O(n)

快速排序 O(logn)

希尔排序 O(1)
